## Print 예제 활용 

	- 만약 "프린트"라는 글이 나오길 원한다면
	- Print "프린트", print'프린트' 혹은 print("프린트"), print('프린트')
	- 라고 작성하면, 4가지다 프린트 라고 나온다.

```python
# -*- coding: utf-8 -*-
"""
Created on Tue Sep 18 16:25:52 2018
@author: HDC_USER
"""
	print "프린트"

	print '프린트'

	print ("프린트")

	print ('프린트')
```

	- 결과는  프린트 프린트 프린트 프린트 로 나온다.
